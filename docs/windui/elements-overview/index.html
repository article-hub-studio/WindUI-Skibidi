Elements Overview

WindUI Modded now documents the element set by purpose instead of putting every control into one long mixed list.

Core controls

ElementUse it for
ButtonRun actions from a compact control.
ToggleEnable or disable a script feature.
SliderPick a numeric value quickly.
StepperMobile-friendly numeric changes with minus and plus controls.
InputSingle-line text values.
TextAreaMulti-line text input.
DropdownPick from a list with fixed mobile positioning.

Selection groups

ElementUse it for
Segmented ControlSwitch between short modes.
Radio GroupChoose one option from a visible set.
Checkbox GroupChoose multiple options.
Chip ListFilter tags and small presets.
ColorpickerPick colors for themes or features.
KeybindBind keyboard and compact mobile actions.

Display and status

ElementUse it for
ParagraphStatic text or media notes.
CalloutImportant contextual messages.
BadgeSmall labels and state chips.
StatusCardServer, feature or runtime status.
StatCardMetrics such as FPS, ping or player count.
KeyValueDense config summaries.
ActionListCommand rows with actions.
MeterGroupMultiple progress values in one block.
DiscordCardA dedicated Discord invite card with copy and join actions.
CardVisual page cards that open dedicated tabs, with optional nested content.

Layout and motion

ElementUse it for
SectionsOrganize content into collapsible groups.
TabSectionSplit content inside a page.
TabBoxMulti-tab content inside a section box.
GroupVisually combine related controls.
HStack and VStackBuild rows and columns inside examples or custom panels.
TimelineShow events and progress history.
AccordionHide secondary details until needed.
EmptyStateShow an intentional blank state with action guidance.
Path2DDraw animated 2D paths.

Media and utility

ElementUse it for
ImageDisplay image previews.
VideoDisplay .webm or Roblox video media.
ViewportRender 3D objects in UI.
ProgressBarShow one progress value.
CodeDisplay code snippets.
DividerSeparate dense sections.
SpaceAdd deliberate spacing when needed.

TabBox

Create nested tabs inside a section or tab.

Section

Organize elements into collapsible sections in WindUI Modded
