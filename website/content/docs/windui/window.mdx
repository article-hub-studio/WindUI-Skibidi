## Dynamic Island Open Button

`EditOpenButton` returns a controller. Its width, height, icon, title, and content animate between four states without recreating the UI. After `AutoHide` seconds, the button contracts into the small `Idle` pill while remaining tappable. When `MorphWindow` is enabled, closing the window keeps its layout intact while the whole surface scales down and travels into the Dynamic Island; opening reverses the same animation.

```luau
local Island = Window:EditOpenButton({
    Title = "WindUI",
    Icon = "geist:sparkles",
    State = "Compact",
    AutoHide = 4,
    IdleWidth = 78,
    IdleHeight = 28,
    Shadow = true,
    ShadowBlur = 18,
    MorphWindow = true,
    MorphDuration = 0.46,
    OnlyMobile = false,
})

Island:Expand({
    Title = "Download complete",
    Content = "All files are ready",
    Icon = "solar:check-circle-bold",
}, 3) -- returns to Compact after 3 seconds

Island:Collapse()       -- icon-only circle
Island:Compact()        -- icon + title pill
Island:Idle()           -- small black pill; Hide() is an alias
Island:Wake()           -- return to Compact and restart AutoHide
Island:ToggleExpanded()
Island:Push({ Title = "Saved", Content = "Config updated" }, 2)
print(Island:GetState())
```

The same operations are available directly on the window:

```luau
Window:ExpandOpenButton({ Title = "Working", Content = "Please wait" })
Window:CompactOpenButton()
Window:CollapseOpenButton()
Window:HideOpenButton()
Window:WakeOpenButton()
Window:SetOpenButtonState("Expanded", { Content = "Custom state" })
```

## Advanced Linked Corners

Linked corners use Roblox's native per-corner `UICorner` radii, so every element surface—not only the outer container—receives the correct top-left, top-right, bottom-left and bottom-right shape. Sequences can split by group or explicit breaks, and HStack automatically switches linking direction when it stacks vertically.

```luau
local Window = WindUI:CreateWindow({
    Title = "Linked UI",
    NewElements = true,
    LinkElementCorners = true,
    CornerLink = {
        InnerRadius = 0,
        Gap = 1,
        BridgeHidden = true,
        BridgeSparse = false,
        BreakTypes = { "Divider", "Space", "Section" },
    },
})

Tab:Button({ Title = "First", CornerGroup = "account" })
Tab:Button({ Title = "Middle", CornerGroup = "account" })
Tab:Button({ Title = "Last", CornerGroup = "account", CornerBreakAfter = true })
Tab:Button({ Title = "Separate", LinkCorners = false })
```

Tabs and containers such as `Group`, `HStack`, and `VStack` may define their own `CornerLink` table. `InnerRadius = 0` uses native per-corner radii to keep every shared seam completely flat. `Gap = 0` joins surfaces edge-to-edge and `Gap = 1` leaves a crisp separator. Individual elements support `CornerGroup`, `CornerBreak`, `CornerBreakBefore`, `CornerBreakAfter`, and `LinkCorners = false`.

## Background API

`SetBackgroundImage` still works, but new code should use `SetBackground`.
It accepts colors, gradients, images, custom assets and `.webm` videos.

```luau
Window:SetBackground("#101821")

Window:SetBackground(WindUI:Gradient({
    ["0"] = { Color = "#071018", Transparency = 0.05 },
    ["100"] = { Color = "#12352F", Transparency = 0.45 },
}, {
    Rotation = 118,
}), 0.48)

Window:SetBackground({
    Image = "https://example.com/panel.png",
    Transparency = 0.35,
    ScaleType = Enum.ScaleType.Crop,
})

Window:SetBackground({
    Video = "https://example.com/background.webm",
    Volume = 0,
    Looped = true,
})

Window:SetBackground("customasset:WindUI/MyHub/assets/background.png")
```

## Loading API

Use the standalone loading API when the loading screen should live outside `CreateWindow`.
`LoadingScreen` in the window config still works for older scripts.

```luau
local Loader = WindUI:LoadingCreate({
    Title = "WindUI",
    Desc = "Preparing interface",
    Icon = "sparkles",
    Steps = { "Theme", "Motion", "Elements" },
    Duration = 5,
    AutoStatus = true,
    CompleteStatus = "Ready",
    CloseDelay = 0.18,
})

-- Duration starts automatic progress and closes after exactly five seconds.
-- Manual control remains available:
Loader:SetProgress(0.35):SetStatus("Loading theme")
Loader:Step(2, "Building UI")
Loader:Play(5)
```
